## Pipeline summary:


For a multiplex input must to add a file **genome_size.csv**  with size of each genome (bp) per barcode. :

e.g
```
barcode,genome_size
01,3000000
02,4500000
03,5000000
04,4000000
05,3200000
06,3500000

```
